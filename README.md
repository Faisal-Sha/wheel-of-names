#### Problem Statement:  
Challenge Wheel Coding Challenge: https://codingchallenges.fyi/challenges/challenge-wheel

#### Steps to Build the Application 

```shell
$ cd challenge-wheel
$ mkdir build 
$ cd build 
$ cmake .. 
$ make 
$ cp -r ../ui/*.* . 
$ ./wheel 
```
